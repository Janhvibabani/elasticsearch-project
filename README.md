## Overview

This project was built to gain a deeper understanding of Elasticsearch and how it compares with traditional database search.

I used the TMDB Movies dataset:
https://www.kaggle.com/datasets/tmdb/tmdb-movie-metadata

The same dataset was loaded into both PostgreSQL and Elasticsearch. I then performed keyword-based queries across both systems to compare:

- Query latency (speed)
- Result relevance
- Full-text search capabilities

This helped highlight the differences between relational search and distributed search engines in real-world scenarios.
